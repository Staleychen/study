# Study Repository

This repository demonstrates a simple structural analysis of a king-post roof truss using the matrix-displacement method.

## Files
- `king_post_analysis.py` — Python script performing the analysis and plotting the truss deformation.

## Usage
Run the script with Python 3 and `matplotlib` installed:

```bash
python3 king_post_analysis.py
```

The program prints joint displacements, support reactions and axial forces, then shows a plot of the undeformed and deformed truss shapes.
